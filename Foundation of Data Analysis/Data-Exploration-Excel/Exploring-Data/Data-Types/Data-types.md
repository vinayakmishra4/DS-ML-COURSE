# 📊 Data Types in Microsoft Excel

Microsoft Excel works with **four primary data types**:

* 🔢 **Numbers**
* 📝 **Text**
* 📅 **Date & Time**
* ✅ **Boolean (Logical Values)**

Understanding these data types is essential because they determine how data is **stored, processed, and analyzed** using formulas and functions.

---

## ⭐ Best Practices When Working with Data Types

✔️ Always verify imported data types
✔️ Apply proper cell formatting *before* entering data
✔️ Avoid mixing numbers and text in calculation columns
✔️ Validate entries using **Data Validation tools**
✔️ Keep data consistent within columns

---

# 📌 1. 📝 Text (String) Data Type

The **Text** data type includes any **non-numeric information** such as names, codes, or descriptions.

### 🔎 Examples

* Names: `John Doe`
* Employee IDs: `EID12345`
* Job Roles: `Software Engineer`
* Product Descriptions: `Wireless Mouse`

### 📍 Default Alignment

➡️ Left-aligned in cells

### 💼 Business Usage

* Customer information storage
* Product catalog management
* Invoice and reference numbers
* Employee records

---

# 📌 2. 🔢 Numeric Data Type

The **Numeric** data type includes values used in **mathematical calculations**.

### 🔎 Examples

* Salaries: `23000`
* Product Prices: `500`
* Quantities: `100`
* Percentages: `25%`
* Currency: `₹15,000`
* Decimal Values: `45.75`

### 📍 Default Alignment

➡️ Right-aligned in cells

### 💼 Usage

* Revenue calculation
* Sales performance tracking
* Financial reporting
* Inventory quantity management
* Budget analysis

---

# 📌 3. ✅ Boolean Data Type

The **Boolean** data type represents logical values:

* `TRUE`
* `FALSE`

These values are commonly generated using logical formulas.

### 🔧 Common Functions Used

* `IF`
* `AND`
* `OR`
* `NOT`

### 💼 Usage

* Checking if sales targets are achieved
* Validating conditions in datasets
* Automating decision-based logic
* Performance tracking dashboards

---

# 📌 4. 📅 Date and Time Data Type

The **Date and Time** data type helps track time-based events and perform chronological calculations.

### 🔎 Examples

* Date: `01/01/2025`
* Time: `10:30 AM`
* Date & Time: `01/01/2025 10:30 AM`

### 🎨 Formatting Options

* Short Date
* Long Date
* Time Only
* Custom Date Formats

### 💼 Usage

* Sales trend tracking by date
* Attendance monitoring
* Project deadline tracking
* Financial period reporting
* Time-based performance analysis

---

Correct data types ensure:

* ✔️ Accurate calculations
