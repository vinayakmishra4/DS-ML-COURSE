## 🚀 Project Overview

This section focuses on **analyzing data using Microsoft Excel**, covering essential techniques such as:

- 📌 Data Organization  
- 📌 Logical Decision Making  
- 📌 Calculation Functions  
- 📌 Count-Based Analysis  

The goal is to transform **raw data into meaningful insights** using Excel functions and tools.

## 📂 Modules Covered

### 🔹 1. Excel Calculation Functions

**Key Functions:**
- IF → Logical decision making

- COUNTA → Count non-empty cells

- VLOOKUP → Lookup values  
- COUNTIFS → Multi-condition counting  
- SUMPRODUCT → Advanced calculations
