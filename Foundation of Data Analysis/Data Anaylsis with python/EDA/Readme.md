## üßê What is EDA?

**Exploratory Data Analysis (EDA)** is the process of analyzing and visualizing datasets to:

* Summarize key characteristics
* Detect anomalies or missing values
* Understand relationships between variables
* Prepare data for modeling

Think of it as **getting to know your data before making decisions**.
*(Source: [Wikipedia](https://en.wikipedia.org/wiki/Exploratory_data_analysis))*
