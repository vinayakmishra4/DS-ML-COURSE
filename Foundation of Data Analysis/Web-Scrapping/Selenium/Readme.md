# ⚙️ Setting Up Selenium & Web Drivers

## 📌 Introduction

Selenium is a powerful automation tool used for web scraping, testing, and automating browser actions. It helps reduce manual effort by interacting with websites just like a real user.

---

## 🧰 Prerequisites

Before getting started, ensure you have:

* Python installed on your system
* Basic understanding of Python

---

## 📥 Installation

Selenium can be installed using package managers like pip. Once installed, it can be imported into your Python environment to start automation tasks.

---

## 🌐 WebDriver Overview

WebDriver is the core component that allows Selenium to control web browsers. It acts as a bridge between your script and the browser, enabling automation of actions like opening pages, clicking elements, and navigating websites.

---

## 🚀 Running a Browser

After setting up WebDriver, Selenium can launch a browser instance (such as Chrome, Firefox, etc.) and begin executing automated tasks.

---

## 📥 ChromeDriver Setup

To use browsers like Chrome, a compatible WebDriver (e.g., ChromeDriver) must be installed. This driver ensures proper communication between Selenium and the browser. Some tools can automate this setup process to make it easier.

---

## 🌍 Opening a Website

Once the browser is launched, Selenium can navigate to any website by providing the desired URL, allowing further interaction and data extraction.

---

## 🧪 Extracting Page Content

Selenium can retrieve the full page source of a website, which can then be parsed using tools like BeautifulSoup to extract meaningful information.
