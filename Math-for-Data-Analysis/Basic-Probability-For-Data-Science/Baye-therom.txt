Q2. Direct Bayes Application
Given:
P(A) = 0.4
P(B|A) = 0.5
P(B) = 0.2

Find P(A|B).

Answer:
P(A|B) = (P(B|A) × P(A)) / P(B)
= (0.5 × 0.4) / 0.2
= 0.2 / 0.2
= 1

Q3. Medical Test Problem
A disease affects 2% of the population. A test has:

* 95% sensitivity (P(+|D))
* 10% false positive rate (P(+|¬D))

If a person tests positive, what is the probability they actually have the disease?

Answer:
P(+) = (0.95 × 0.02) + (0.10 × 0.98)
= 0.019 + 0.098
= 0.117

P(D|+) = 0.019 / 0.117 ≈ 0.162

Final Answer ≈ 16.2%

Q4. Machine Defect Problem
A factory has two machines:

* Machine A produces 60% of items with a 3% defect rate
* Machine B produces 40% of items with a 5% defect rate

If a randomly selected item is defective, find the probability it was produced by Machine A.

Answer:
P(D) = (0.6 × 0.03) + (0.4 × 0.05)
= 0.018 + 0.020
= 0.038

P(A|D) = (0.6 × 0.03) / 0.038
= 0.018 / 0.038 ≈ 0.474

Final Answer ≈ 47.4%

Q5. Multiple Boxes Problem
There are three boxes:

* Box 1: 2 red balls, 1 blue ball
* Box 2: 1 red ball, 2 blue balls
* Box 3: 3 red balls, 0 blue balls

A box is chosen at random, and a red ball is drawn.
Find the probability that the ball came from Box 3.

Answer:
P(R) = (1/3 × 2/3) + (1/3 × 1/3) + (1/3 × 1)
= 2/9 + 1/9 + 3/9
= 6/9 = 2/3

P(Box 3 | R) = (1 × 1/3) / (2/3)
= 1/2

Final Answer = 0.5
