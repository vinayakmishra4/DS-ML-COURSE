## ✨ Why Choose This Course?

Breaking into Data Science requires more than just knowing theoretical math—it requires **building**. This repository is engineered to take you step-by-step from writing your first line of Python to deploying complex machine learning models. 

Every module is paired with real-world projects so you understand not just *how* the code works, but *why* it solves critical business problems.

### 🎯 Who is this for?
* **Beginners** looking for a structured, no-nonsense roadmap.
* **Analysts** wanting to upgrade their skills with Python and automation.
* **Engineers** transitioning into AI, Data Science, and Machine Learning.

## 🚀 Module 2 Projects: Applied Data Science

A portfolio of real-world datasets and automation scripts demonstrating practical data extraction and analysis.

### 📈 Exploratory Data Analysis (EDA)

> **[🌾 Sugarcane Production Analysis](https://www.google.com/search?q=https://github.com/vinayakmishra4/Project-Sugarcane-Production)**
> Analyzed agricultural production trends across regions focusing on data cleaning, visualization, and actionable insights.

> **[🛒 Black Friday Sales Analysis](https://www.google.com/search?q=https://github.com/vinayakmishra4/Project-Black-Firday-Sales-Analysis)**
> Investigated customer behavior and retail patterns to identify top products and demographic purchasing trends.

> **[❤️ Heart Disease Data Analysis](https://www.google.com/search?q=https://github.com/vinayakmishra4/Project-Heart-Disease-Analysis)**
> Performed in-depth EDA on medical records to visualize health patterns and study critical heart disease risk factors.

> **[🌍 Global GDP Analysis](https://www.google.com/search?q=https://github.com/vinayakmishra4/Project-Gross-Domestic-Product-Analysis)**
> Mapped historical GDP data to understand global economic shifts and perform statistical country comparisons.

### 🕷️ Data Engineering & Web Automation

> **[🎥 YouTube Scraping & Data Analysis](https://github.com/vinayakmishra4/Project-Youtube-Scrapping)**
> An end-to-end Python pipeline using `Selenium` to scrape video metadata, clean shorthand metrics (e.g., converting "1.5M" to `1500000`) with Pandas, and generate visual insights.

> **[📸 Stock Image Scraper](https://www.google.com/search?q=https://github.com/vinayakmishra4/Stock-Image-Scraper-)**
> An automation tool bypassing infinite scroll with `Selenium` to download high-res images directly to local drives and export metadata to a Pandas-ready Excel dataset.

> **[🌐 Wikipedia Scraper](https://www.google.com/search?q=https://github.com/vinayakmishra4/Project-Wikipedia-Scrapper/blob/main/README.md)**
> Uses `requests` and `BeautifulSoup` to search Google for a person's Wikipedia page (automatically handling misspellings) and extracts clean paragraph text into a `.txt` file.

> **[📚 Book Scraper](https://www.google.com/search?q=https://github.com/your-username/book-scraper)**
> A targeted scraper collecting book titles, prices, ratings, and availability, formatting the output into a structured CSV for downstream analysis.
