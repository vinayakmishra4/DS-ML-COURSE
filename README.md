## ✨ Why This Course?

This repository is designed to take you from **zero coding knowledge to a job-ready Data Scientist**. Instead of just theory, this journey is built on **hands-on coding, real datasets, and practical projects**. Each module systematically builds your skills, ensuring you understand not just *how* to write code, but *why* it works in a real-world data pipeline.

## 🚀 Module 2 Projects: Applied Data Science

Real-world datasets and automation scripts demonstrating practical data analysis and extraction.

### 📈 Exploratory Data Analysis (EDA)
* **[🌾 Sugarcane Production Analysis](https://github.com/vinayakmishra4/Project-Sugarcane-Production):** Analyzed agricultural production trends across regions focusing on data cleaning, visualization, and actionable insights.
* **[🛒 Black Friday Sales Analysis](https://github.com/vinayakmishra4/Project-Black-Firday-Sales-Analysis):** Investigated customer behavior and retail patterns to identify top products and demographic trends.
* **[❤️ Heart Disease Data Analysis](https://github.com/vinayakmishra4/Project-Heart-Disease-Analysis):** Performed EDA on medical data to visualize health patterns and study heart disease risk factors.
* **[🌍 Global GDP Analysis](https://github.com/vinayakmishra4/Project-Gross-Domestic-Product-Analysis):** Analyzed historical GDP trends to understand global economic patterns and country comparisons.

### 🕷️ Web Scraping & Automation pipelines
* **[🎥 YouTube Scraping & Data Analysis](https://github.com/vinayakmishra4/Project-Youtube-Scrapping):** An end-to-end Python pipeline using `Selenium` to scrape video metadata, clean shorthand metrics (e.g., "1.5M") with `Pandas`, and generate visual insights with `Matplotlib`.
* **[📸 Stock Image Scraper](https://github.com/vinayakmishra4/Stock-Image-Scraper-):** An automation tool bypassing infinite scroll with `Selenium` to download high-res images directly to local drives and export metadata to a Pandas-ready Excel dataset.
* **[🌐 Wikipedia Scraper](https://github.com/vinayakmishra4/Project-Wikipedia-Scrapper/blob/main/README.md):** Uses `requests` and `BeautifulSoup` to search Google for a person's Wikipedia page (handling misspellings) and extracts clean paragraph text into a `.txt` file.
* **[📚 Book Scraper](https://github.com/your-username/book-scraper):** A targeted scraper collecting book titles, prices, ratings, and availability, formatting the output into a structured CSV. *(Note: Update this link with your correct repo URL!)*
